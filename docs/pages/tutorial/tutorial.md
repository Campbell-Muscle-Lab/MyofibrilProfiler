# Tutorial
{:.no_toc}

* TOC
{:toc}

Chekhov has multiple features and uses a multi window approach to provide desired features:

- Chekhov: Main app window for users to generate signals and collect data.
- ChekhovPockelCellCalibration: App window to calibrate Pockel cell. This step needs to be completed beginning of every experiment.
- ChekhovCamera: App window to access the camera feed of the experimental setup. 

Chekhov is currently configured to work with National Instruments data acquisition boards and Thorlabs cameras. While a small tweaking is needed to work with it other hardware, it is compatible with all the models from above manufacturers.

The getting started tutorial includes how to use the all of the parts of the software.
